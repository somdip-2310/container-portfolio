Frequently Asked Questions

Deploying your first container is easy:

  1. Click the "Deploy Your Application" button on the Containers page
  2. Enter your container name and select a Docker image
  3. Configure the port and environment variables (optional)
  4. Click "Deploy" and wait for your container to start

Your container will be accessible via the provided URL once deployment completes.

To add a custom domain:

  1. Navigate to the Domains page
  2. Click "Add Domain" and enter your domain name
  3. Select the container you want to connect
  4. Create a CNAME record at your DNS provider pointing to the provided target
  5. Wait for DNS propagation (5-60 minutes)

SSL certificates are automatically provisioned once verification is complete.

You can view container logs in two ways:

  • From Container Card: Click the "Logs" button on any container card
  • From Logs Page: Visit the Logs page and select a container from the dropdown

You can filter logs by keyword, adjust the number of lines shown, and download logs for offline analysis.
